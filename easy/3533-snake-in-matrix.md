Easy

## Problem Description
<p>There is a snake in an <code>n x n</code> matrix <code>grid</code> and can move in <strong>four possible directions</strong>. Each cell in the <code>grid</code> is identified by the position: <code>grid[i][j] = (i * n) + j</code>.</p>

<p>The snake starts at cell 0 and follows a sequence of commands.</p>

<p>You are given an integer <code>n</code> representing the size of the <code>grid</code> and an array of strings <code>commands</code> where each <code>command[i]</code> is either <code>&quot;UP&quot;</code>, <code>&quot;RIGHT&quot;</code>, <code>&quot;DOWN&quot;</code>, and <code>&quot;LEFT&quot;</code>. It&#39;s guaranteed that the snake will remain within the <code>grid</code> boundaries throughout its movement.</p>

<p>Return the position of the final cell where the snake ends up after executing <code>commands</code>.</p>

## Solution
1. Initial Thoughts:
   - The problem describes a snake moving on an n x n grid based on a sequence of commands.
   - The snake's position is represented by a single integer calculated as `(i * n) + j`, where `i` is the row and `j` is the column.
   - We need to simulate the snake's movements based on the commands and return its final position.
   - Since the snake stays within the grid boundaries, we don't need to worry about out-of-bounds checks explicitly.

2. Solution Implementation:
```python
def find_final_position(n, commands):
    """
    Finds the final position of the snake in a matrix after executing a sequence of commands.

    Args:
        n: The size of the n x n matrix.
        commands: A list of strings representing the movement commands.

    Returns:
        The final position of the snake.
    """

    row, col = 0, 0  # Initial position of the snake (cell 0)
    position = 0

    for command in commands:
        if command == "UP":
            row -= 1
        elif command == "DOWN":
            row += 1
        elif command == "LEFT":
            col -= 1
        elif command == "RIGHT":
            col += 1

    position = (row * n) + col
    return position


```

3. Solution Explanation:
   - We initialize the snake's position at `row = 0` and `col = 0` (cell 0).
   - We iterate through the `commands` list.
   - Inside the loop, we update the `row` and `col` based on the current command ("UP", "DOWN", "LEFT", or "RIGHT").
   - After processing all commands, we calculate the final position using the formula `position = (row * n) + col` and return it.


4. Complexity Analysis:
   - Time complexity: O(m) - where m is the length of the `commands` list. We iterate through the commands once.
   - Space complexity: O(1) - We use constant extra space to store the `row`, `col`, and `position` variables. The space used doesn't depend on the input size.
